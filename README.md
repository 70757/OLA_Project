# OLA_Project
Ola Ride Data Analysis project using SQL, Power BI

##  Overview  
This project focuses on analyzing **ride cancellation trends** and **behavioral patterns** of customers and drivers using Ola ride data.  
The dataset was cleaned and processed using **SQL** to extract meaningful insights, and **Power BI** was used to visualize key factors influencing cancellations.

---

## SQL Analysis  
A series of SQL queries were written to:
- Aggregate data by ride status, driver ratings, and vehicle types  
- Identify **peak hours** and **common cancellation reasons**  
- Explore **customer and driver behavior** patterns affecting ride outcomes  

---

## Power BI Dashboard  
Created insightful dashboards to visualize:
- Ride cancellation trends over time and by location  
- Correlations between **driver ratings** and **cancellation rates**  
- Booking preferences and **payment method insights**

---

##  Key Insights  
- **Peak-hour cancellations** and **specific vehicle types** showed higher cancellation rates  
- **Driver ratings** had a clear correlation with cancellations  
- **Payment methods** and **pickup location issues** were among the top cancellation factors  

After implementing targeted strategies, overall cancellation rates decreased by **10%**

---

##  Strategy Highlights  
- Improved **driver availability** during high-demand hours  
- Enhanced **customer experience** with clearer booking options and flexible payments
